# Travel Agent Benchmark

## Overview

The Travel Agent Benchmark is a collection of evaluation scenarios designed to measure the performance of AI-powered travel planning agents.

Rather than evaluating a single response subjectively, the benchmark defines structured scenarios with expected behaviour, evaluation criteria, pass criteria, and failure conditions. This enables travel-planning agents to be evaluated consistently and compared across different models, prompts, and implementations.

## Objective

The benchmark evaluates how effectively a travel-planning agent can:

* Satisfy user constraints.
* Produce high-quality travel plans.
* Provide accurate and reliable information.
* Personalize recommendations based on user preferences.
* Adapt to changing requirements when applicable.

The benchmark will continue to expand with additional scenarios and evaluation profiles for different categories of AI agents. The long-term goal is to provide a reusable benchmark suite that enables consistent evaluation of agent performance across multiple domains.
